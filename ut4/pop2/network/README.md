# UT4-POP2: Modularidad (Objetos y Clases + Excepciones + Módulos)

# 🟪 PUESTO IMPAR

Esta prueba gira en torno al trabajo con **redes**, concretamente de equipos ("host") que se identifican mediante su dirección IP.

## Plantilla

→ [network.py](./templates/network.py)

## Tests

→ [test_network.py](./test_network.py)

## Durante la prueba

- Utiliza únicamente recursos que hayamos visto en clase hasta el momento.
- Lanza los tests con: `pytest test_network.py`
- Puedes parar tras el primer test fallido con: `pytest -x test_network.py`
- Puedes lanzar un test en concreto con: `pytest test_network.py::test_build_host_by_full_str_ip`

## Al finalizar

- Sube únicamente el fichero `network.py`

## Evaluación

- Cada test cuenta 0.5 puntos.
- La nota final corresponderá a la suma de los tests que hayan pasado.
- Hay que tener en cuenta que se puede bajar nota en función de la calidad del código implementado.
- La copia parcial o total supondrá un 0 en la nota global de la prueba.
