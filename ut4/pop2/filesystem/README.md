# UT4-POP2: Modularidad (Objetos y Clases + Excepciones + Módulos)

Esta prueba gira en torno al trabajo con **sistemas de ficheros**.

## Plantilla

→ [filesystem.py](./templates/filesystem.py)

## Tests

→ [test_filesystem.py](./test_filesystem.py)

## Durante la prueba

- Utiliza únicamente recursos que hayamos visto en clase hasta el momento.
- Lanza los tests con: `pytest test_filesystem.py`
- Puedes parar tras el primer test fallido con: `pytest -x test_filesystem.py`
- Puedes lanzar un test en concreto con: `pytest test_filesystem.py::test_is_directory`

## Al finalizar

- Sube únicamente el fichero `filesystem.py`

## Evaluación

- Cada test cuenta 0.5 puntos.
- La nota final corresponderá a la suma de los tests que hayan pasado.
- Hay que tener en cuenta que se puede bajar nota en función de la calidad del código implementado.
- La copia parcial o total supondrá un 0 en la nota global de la prueba.
