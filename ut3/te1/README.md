# UT3-TE1: Aplicación de las estructuras de almacenamiento (Listas)

### TAREA EVALUABLE

![Python sticker](../../python-sticker.png)

## Objetivo

Escriba un programa en Python que permita jugar al **TRES EN RAYA**.

## Características

Características del programa:

- Debe diferenciar los jugadores.
- Debe pedir en cada turno la posición en la que jugar.
- Debe "pintar" el tablero en cada turno.
- Debe "prohibir" que se coloque ficha en una celda ya ocupada.
- Debe "prohibir" que se coloque ficha fuera del tablero.
- Debe indicar si algún jugador gana la partida.
- Debe indicar si la partida termina en empate.

## Notas

- Utilizar sólo herramientas de Python que se hayan visto hasta el momento en clase.
