## Comprimir los ejercicios

Cambia a la carpeta adecuada:

```console
mkdir -p ~/pro/ut6/tee6.1 && cd ~/pro/ut6/tee6.1
```

Descarga y comprime todos los ejercicios que has subido durante este trimestre:

```console
pypas pull t2 && zip -FSr t2.zip t2
```

> Esto genera el archivo `t2.zip` (además de una carpeta `t2/`) con todos tus ejercicios del trimestre.

## Entrega

- Sube el archivo `t2.zip` a la tarea correspondiente del aula virtual.

## Evaluación

- La puntuación final de la prueba estará en función del **número de tests** que se hayan superado.
- Podrá haber penalizaciones por **copia** o **calidad del código**.
